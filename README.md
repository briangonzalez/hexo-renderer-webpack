# hexo-renderer-webpack

Add [Hexo] support for [Webpack].

## Install

``` bash
$ npm install hexo-renderer-webpack --save
```

## Options

You can configure this plugin in `_config.yml` or your theme's `_config.yml`.

``` yaml
webpack:
  entry: 'themes/my-theme/source/js/app.js'
```
